SETUP XLAP / OFFOREST TREN VPS TU GIT
=====================================

Gia dinh:
- VPS Ubuntu 22.04/24.04
- Web server Nginx + PHP-FPM
- PHP 8.3+
- Node.js 20+
- MySQL/MariaDB
- Project path tren VPS: /var/www/xlap
- Domain vi du: https://your-domain.com

Hay thay cac gia tri nay truoc khi chay:
- REPO_URL_HERE
- your-domain.com
- DB_NAME_HERE
- DB_USER_HERE
- DB_PASSWORD_HERE
- TURNSTILE_SITE_KEY_HERE
- TURNSTILE_SECRET_KEY_HERE
- GOOGLE_DRIVE_* neu dung Drive


1. SSH VAO VPS
==============

ssh root@YOUR_VPS_IP


2. CAI GOI HE THONG
===================

apt update
apt install -y software-properties-common ca-certificates curl unzip git supervisor nginx mysql-server

add-apt-repository ppa:ondrej/php -y
apt update

apt install -y php8.3 php8.3-fpm php8.3-cli php8.3-common php8.3-mysql php8.3-mbstring php8.3-xml php8.3-curl php8.3-zip php8.3-bcmath php8.3-gd php8.3-intl php8.3-readline

php -v


3. CAI COMPOSER
===============

cd /tmp
curl -sS https://getcomposer.org/installer -o composer-setup.php
php composer-setup.php --install-dir=/usr/local/bin --filename=composer
composer --version


4. CAI NODE.JS 20
=================

curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
node -v
npm -v


5. CLONE CODE TU GIT
====================

mkdir -p /var/www

git clone REPO_URL_HERE xlap

cd /var/www/xlap


6. TAO DATABASE
===============

mysql

Trong MySQL shell, chay:

CREATE DATABASE DB_NAME_HERE CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'DB_USER_HERE'@'localhost' IDENTIFIED BY 'DB_PASSWORD_HERE';
GRANT ALL PRIVILEGES ON DB_NAME_HERE.* TO 'DB_USER_HERE'@'localhost';
FLUSH PRIVILEGES;
EXIT;

8. CAU HINH CLOUDFLARE TURNSTILE LOGIN
======================================

Neu chua co key hoac dang setup lan dau, tam tat:

TURNSTILE_ENABLED=false
TURNSTILE_SITE_KEY=
TURNSTILE_SECRET_KEY=

Khi domain da tro ve VPS va da tao widget trong Cloudflare Turnstile, bat lai:

TURNSTILE_ENABLED=true
TURNSTILE_SITE_KEY=TURNSTILE_SITE_KEY_HERE
TURNSTILE_SECRET_KEY=TURNSTILE_SECRET_KEY_HERE

Luu y tren Cloudflare Dashboard:
- Vao Turnstile
- Tao widget moi
- Add domain: your-domain.com
- Copy Site Key va Secret Key vao .env

11. SET PERMISSION
==================

chmod -R ug+rwx /var/www/xlap/storage /var/www/xlap/bootstrap/cache


12. CAU HINH NGINX
==================

nano /etc/nginx/sites-available/xlap

Dan noi dung nay, nho sua your-domain.com:

server {
    listen 80;
    listen [::]:80;
    server_name your-domain.com www.your-domain.com;

    root /var/www/xlap/public;
    index index.php index.html;

    client_max_body_size 100M;

    add_header X-Frame-Options "SAMEORIGIN";
    add_header X-Content-Type-Options "nosniff";

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location = /favicon.ico { access_log off; log_not_found off; }
    location = /robots.txt  { access_log off; log_not_found off; }

    error_page 404 /index.php;

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/run/php/php8.3-fpm.sock;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }

    location ~ /\.(?!well-known).* {
        deny all;
    }
}

Luu va chay:

ln -s /etc/nginx/sites-available/xlap /etc/nginx/sites-enabled/xlap
nginx -t
systemctl reload nginx
systemctl restart php8.3-fpm


13. TRO DOMAIN VA BAT SSL
=========================

Tren Cloudflare DNS:
- Tao A record your-domain.com -> YOUR_VPS_IP
- Proxy mau cam: bat neu muon qua Cloudflare
- SSL/TLS mode: Full strict neu VPS co SSL hop le

Cai SSL Let's Encrypt tren VPS:

apt install -y certbot python3-certbot-nginx
certbot --nginx -d your-domain.com -d www.your-domain.com

Sau SSL:

nginx -t
systemctl reload nginx


14. SETUP QUEUE WORKER
======================

nano /etc/supervisor/conf.d/xlap-worker.conf

Dan noi dung:

[program:xlap-worker]
process_name=%(program_name)s_%(process_num)02d
command=php /var/www/xlap/artisan queue:work --sleep=3 --tries=3 --timeout=300
autostart=true
autorestart=true
stopasgroup=true
killasgroup=true
user=www-data
numprocs=1
redirect_stderr=true
stdout_logfile=/var/www/xlap/storage/logs/worker.log
stopwaitsecs=3600

Luu va chay:

supervisorctl reread
supervisorctl update
supervisorctl start xlap-worker:*
supervisorctl status
