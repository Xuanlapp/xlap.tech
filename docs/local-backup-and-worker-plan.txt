LOCAL BACKUP VA LOCAL WORKER CHO VPS XLAP
=========================================

Muc tieu:
- Backup database va folder anh tren VPS ve may local.
- Can biet may local co the nhan job render mockup tu VPS chua.


KET LUAN HIEN TAI
=================

1. Backup ve local: lam duoc ngay.

May local khong can chay 24/24. Chi can chay script backup theo lich, vi du moi ngay 1 lan.

2. Local worker render mockup cho VPS: chua dung duoc ngay voi code hien tai.

Da check code:

app/Livewire/Pages/Sticker/ProductDesignCard.php
app/Livewire/Pages/Ornament/ProductDesignCard.php

Hien tai method generatePsdMockups() dang goi service render truc tiep:

app(StickerService::class)->generatePsdMockups(...)
app(OrnamentService::class)->generatePsdMockups(...)

Nghia la khi user bam nut tren web VPS, VPS tu render PSD bang Node. Local worker se khong nhan duoc viec gi.

Muon local nhan xu ly, can sua code sang queue:

User bam Generate
  -> Web VPS dispatch job vao database queue
  -> Local worker doc jobs trong database VPS
  -> Local worker render PSD
  -> Local worker day anh ve storage cua VPS
  -> Local worker update database VPS


PHAN 1 - BACKUP VPS VE MAY LOCAL
================================

Chay trong PowerShell tren may local Windows.

Sua cac gia tri:
- YOUR_VPS_IP
- Neu VPS SSH khong dung port 22 thi them -P vao scp va -p vao ssh.

Script backup:

$VpsHost = "nhxlap@YOUR_VPS_IP"
$RemoteProject = "/www/wwwroot/xlap.tech"
$BackupRoot = "$env:USERPROFILE\Desktop\xlap-vps-backups"
$Stamp = Get-Date -Format "yyyyMMdd-HHmmss"
$LocalDir = Join-Path $BackupRoot $Stamp

New-Item -ItemType Directory -Force -Path $LocalDir | Out-Null

$RemoteScript = @'
cd /www/wwwroot/xlap.tech || exit 1
mkdir -p storage/app/backups

DB_DATABASE=$(grep '^DB_DATABASE=' .env | cut -d '=' -f2- | sed 's/^"//;s/"$//')
DB_USERNAME=$(grep '^DB_USERNAME=' .env | cut -d '=' -f2- | sed 's/^"//;s/"$//')
DB_PASSWORD=$(grep '^DB_PASSWORD=' .env | cut -d '=' -f2- | sed 's/^"//;s/"$//')

mysqldump --single-transaction --quick -u"$DB_USERNAME" -p"$DB_PASSWORD" "$DB_DATABASE" | gzip > "storage/app/backups/db-${STAMP}.sql.gz"
tar -czf "storage/app/backups/storage-public-${STAMP}.tar.gz" storage/app/public
tar -czf "storage/app/backups/env-${STAMP}.tar.gz" .env
'@

$RemoteScript | ssh $VpsHost "STAMP=$Stamp bash -s"

scp "${VpsHost}:${RemoteProject}/storage/app/backups/db-${Stamp}.sql.gz" $LocalDir
scp "${VpsHost}:${RemoteProject}/storage/app/backups/storage-public-${Stamp}.tar.gz" $LocalDir
scp "${VpsHost}:${RemoteProject}/storage/app/backups/env-${Stamp}.tar.gz" $LocalDir

ssh $VpsHost "rm -f ${RemoteProject}/storage/app/backups/db-${Stamp}.sql.gz ${RemoteProject}/storage/app/backups/storage-public-${Stamp}.tar.gz ${RemoteProject}/storage/app/backups/env-${Stamp}.tar.gz"

Write-Host "Backup saved to: $LocalDir"


Ghi chu bao mat:
- File env backup co secret. Nen luu trong o dia co BitLocker hoac nen ma hoa file backup.
- Khong upload backup len noi cong khai.
- Backup code khong can thiet neu code da nam tren git. Quan trong nhat la database, storage/app/public, va .env.


PHAN 2 - RESTORE BACKUP LEN VPS
===============================

Copy backup len VPS:

scp "C:\Users\Admin\Desktop\xlap-vps-backups\YYYYMMDD-HHMMSS\db-YYYYMMDD-HHMMSS.sql.gz" nhxlap@YOUR_VPS_IP:/tmp/
scp "C:\Users\Admin\Desktop\xlap-vps-backups\YYYYMMDD-HHMMSS\storage-public-YYYYMMDD-HHMMSS.tar.gz" nhxlap@YOUR_VPS_IP:/tmp/
scp "C:\Users\Admin\Desktop\xlap-vps-backups\YYYYMMDD-HHMMSS\env-YYYYMMDD-HHMMSS.tar.gz" nhxlap@YOUR_VPS_IP:/tmp/

Tren VPS:

cd /www/wwwroot/xlap.tech

tar -xzf /tmp/env-YYYYMMDD-HHMMSS.tar.gz -C /www/wwwroot/xlap.tech
tar -xzf /tmp/storage-public-YYYYMMDD-HHMMSS.tar.gz -C /www/wwwroot/xlap.tech

DB_DATABASE=$(grep '^DB_DATABASE=' .env | cut -d '=' -f2- | sed 's/^"//;s/"$//')
DB_USERNAME=$(grep '^DB_USERNAME=' .env | cut -d '=' -f2- | sed 's/^"//;s/"$//')
DB_PASSWORD=$(grep '^DB_PASSWORD=' .env | cut -d '=' -f2- | sed 's/^"//;s/"$//')

gunzip -c /tmp/db-YYYYMMDD-HHMMSS.sql.gz | mysql -u"$DB_USERNAME" -p"$DB_PASSWORD" "$DB_DATABASE"

php artisan optimize:clear
php artisan storage:link
sudo chown -R www:www storage bootstrap/cache
sudo chmod -R ug+rwx storage bootstrap/cache


PHAN 3 - LOCAL CAN CHAY GI DE NHAN XU LY VPS
============================================

Chi can lam phan nay sau khi code da duoc sua sang queue job.

May local can chay 3 thu:

1. SSH tunnel toi database VPS.
2. Queue worker Laravel.
3. Cach day anh render xong ve storage cua VPS.


3.1 SSH TUNNEL DATABASE
-----------------------

Khong mo MySQL 3306 public ra internet.

Chay PowerShell/terminal local:

ssh -N -L 3307:127.0.0.1:3306 nhxlap@YOUR_VPS_IP

Cua so nay phai giu mo trong luc local worker chay.


3.2 .ENV LOCAL WORKER
---------------------

Trong project local worker, .env can tro ve database VPS qua tunnel:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3307
DB_DATABASE=ten_database_vps
DB_USERNAME=user_database_vps
DB_PASSWORD=password_database_vps

QUEUE_CONNECTION=database
FILESYSTEM_DISK=local
APP_URL=https://xlap.tech

Mockup renderer local:

PSD_MOCKUP_RENDERER_COMMAND="node scripts/psd-renderer/render.js"
OFFOREST_TRIM_MOCKUP_DESIGN=false
OFFOREST_MOCKUP_DESIGN_SCALE=0.72


3.3 CHAY QUEUE WORKER TREN LOCAL
--------------------------------

PowerShell local:

Set-Location -LiteralPath "I:\XLAP\Web\VPS\xlap"
php artisan queue:work --tries=1 --timeout=900

Nhung lenh nay chi co tac dung khi web VPS da dispatch job vao bang jobs.


3.4 VAN DE STORAGE
------------------

Neu local render ra file trong local storage/app/public thi VPS khong thay anh.

Co 2 cach:

Cach A - Sau render, code local copy file ve VPS bang scp/rsync.

VPS path:

/www/wwwroot/xlap.tech/storage/app/public/generated/...

DB van luu:

/storage/generated/...

Cach B - Mount folder VPS storage/app/public vao local.

Local storage/app/public thuc chat tro toi:

VPS:/www/wwwroot/xlap.tech/storage/app/public

Cach B tien nhung phu thuoc ket noi mang. Cach A de kiem soat hon.


PHAN 4 - CAN SUA CODE NHU THE NAO
=================================

Can them job, vi du:

app/Jobs/GenerateStickerPsdMockups.php
app/Jobs/GenerateOrnamentPsdMockups.php

Job implements ShouldQueue.

ProductDesignCard::generatePsdMockups() khong goi render truc tiep nua, ma dispatch job:

GenerateStickerPsdMockups::dispatch(auth()->id(), $this->assetId);

Job se goi:

app(StickerService::class)->generatePsdMockups($user, $assetId);

Tuong tu Ornament.

Sau do UI hien toast:

"Da dua vao hang doi render mockup."

Va card can refresh/poll de thay anh khi worker xu ly xong.


PHAN 5 - BAO MAT CHO LOCAL WORKER
=================================

- Khong mo MySQL public.
- Dung SSH tunnel.
- Dung SSH key thay password.
- Tao user rieng tren VPS neu local can upload file.
- User upload chi duoc ghi vao storage/app/public/generated, khong duoc ghi vao code app.
- Firewall VPS chi mo 80, 443, SSH.
- Turnstile bat cho login.
- APP_DEBUG=false tren production.
